# MAX7219 HACS Card

This repository now includes a Lovelace custom card for controlling the MAX7219
MQTT add-on.

## Card File

- `hacs/max7219-control-card.js`

## HACS Metadata

- `hacs.json`

## Card Type

- `custom:max7219-control-card`

## Default Namespace

- `mirarus/max7219`

You can override namespace in card config:

```yaml
type: custom:max7219-control-card
title: MAX7219 Display
namespace: mirarus/max7219
```
